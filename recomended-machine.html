推奨環境

カルステップの動作に必要な
システム環境について

基本システム要件

対応OS

  • Windows 10 (64bit) 推奨
  • Windows 11 (64bit) 推奨
  • ※Windows 8.1以前の動作保証はありません

CPU

  • 推奨： Intel Core i5 以上
  • 最低： Intel Core i3 以上
  • AMD Ryzen 5 以上も推奨

メモリ

  • 推奨： 8GB
  • 最低： 4GB

インターネット接続要件

必須接続環境

  • 常時インターネット接続（有線・無線問わず）
  • 推奨速度： 下り10Mbps
  • 安定したネットワーク環境

セキュリティ要件

  • HTTPS通信対応
  • Microsoft Azure（*.azure.com）へのアクセス許可
  • Google Cloud（*.googleapis.com）へのアクセス許可
  • ファイアウォール設定での許可が必要な場合があります

電子カルテシステム対応状況

クラウド型電子カルテ

最適

インターネット経由でアクセスするクラウド型電子カルテとの相性が最も良好です。

オンプレミス型電子カルテ

一部対応可

院内サーバー型でもインターネット接続があれば使用可能です。

紙カルテ

対応可

紙カルテ運用でも、パソコン併用で診察効率化が可能です。

  • AI要約文の印刷・貼付
  • 音声録音による記録補完
  • 院内デジタル化の第一歩

推奨ハードウェア構成

標準構成

  • CPU: Intel Core i5 / AMD Ryzen 5
  • メモリ: 8GB
  • OS: Windows 10 Home以上

重要な注意事項

非対応環境

  • Mac OS（macOS）
  • Linux系OS
  • 32bit版Windows
  • インターネット接続のない環境

※Windows 8.1以前は動作保証外です。
